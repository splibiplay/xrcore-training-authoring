Editor-first authoring module for XR training scenario creation and generation.

## GitHub policy

This repository is maintained as **documentation, manuals, and integration guidance**.
Runtime/editor implementation is distributed via Unity package channels.

## Published foundation (Unity Asset Store)

Install XRCore foundation first:

- [XRCore - Event-Driven AI Framework for Unity XR](https://assetstore.unity.com/packages/tools/ai-ml-integration/xrcore-event-driven-ai-framework-for-unity-xr-366852)

## What this module adds

- visual scenario + step authoring workflows
- completion conditions and validator assignment
- generated runtime-ready assets
- generated playable demo scene
- preflight, smoke, and release-oriented checks

## Quick start

1. Open `Tools -> XRCore -> General -> Authoring -> Setup Wizard`.
2. Open `Training Configurator`.
3. Define scenario and steps.
4. Run preflight gate.
5. Build package and generate demo scene.
